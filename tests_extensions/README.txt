--------------------------------------------
Vorbereitende Tests
============================================

--------------------------------------------
1) Wurzel
--------------------------------------------
input H-PILoT	sqrt_tests.loc
Tests für die Axiomatisierung einer Wurzelrelation, bestehend aus sqrt(x,y), wobei x die Wurzel 
von y ist, unter der Bedingung dass x positiv ist und x*x=y für den Fall dass sqrt(x,y)=_1.

--------------------------------------------
2) Maximum
--------------------------------------------
input H-PILoT	maximum_tests.loc
Tests für die Axiomatisierung des Maximums zweier Zahlen. 

--------------------------------------------
3) Betrag
--------------------------------------------
input H-PILoT	amount_tests.loc
Tests für die Axiomatisierung des Betrags einer Zahl.
